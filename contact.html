Ce projet avait pour but de créer une plateforme afin de gérer les clients d'un garage ainsi que les interventions. Il dispose d'une interface client où l'on peut apercevoir ses véhicules enregistrés ainsi qu'une interface admin qui nous permet d'insérer un nouveau client, son véhicule, un technicien et de créer une intervention qui les relie tous.

GitHub lance son outil Copilot pour aider les développeurs

Github Copilot est une intelligence artificielle créée par GitHub et OpenAI dans le but d'aider les développeurs. Elle est capable de générer du code à partir d'un commentaire mais aussi de faire une prédiction de ce que vous voulez potentiellement écrire grace au modèle de Langages GPT3 développé par OpenAI.

Cette AI permet donc la prédiction dans une douzaine de Langages et s'installe sur certains IDE dont Visual Studio Code. GitHub Copilot est disponible par abonnement mensuel (9.99$) ou annuel (99$) mais dispose aussi d'une offre gratuite pour les étudiants Présenté le 29 juin 2021, GitHub Copilot est maintenant disponible pour les développeurs depuis le 20 juillet 2021. Afin d'apprendre à coder Github Copilot s'est entrainé sur plus de 1 milliard de lignes de code à travers tous les repositorys github disponibles.

Afin de valider la proposition de github il suffit de cliquer sur la touche TAB et de laisser github copilot faire son travail. Attention : Github Copilot est encore en phase de test et peut donc proposer des erreurs de syntaxe ou des erreurs de logique
