Val'auto - Auto-école

Avec mon groupe scolaire composé de Louis Lamouche, Valentio Pinto et moi-même nous sommes actuellement en train de travailler sur un site d'auto-école afin de le présenter le jour de notre épreuve de BTS. Ce projet à pour but de réaliser la vitrine d'une auto-école ainsi qu'une interface utilisateur pour ceux ayant un forfait que l'auto-école. Grâce à cette interface l'utilisateur pourra consulter son forfait, ses heures effectués, son agenda avec ses prochaines heures de conduite, la possibilité de demander ou annuler des heures et avoir accès à un mini code de la route afin de s'entrainer. Le site disposera aussi d'une interface moniteur ou il pourra accepter ou non les demandes de ses élèves ainsi qu'accéder à son agenda. Pour finir il y aura une interface Admin permettant d'insérer ou supprimer un élève et un moniteur

Html, Css, Javascript, Php, Mysql

Bank Dashboard

'Bank Dashboard' est le nom de ma première application web réalisé avec Node Js et certaines librairies. Elle permet de gérer ses revenus et ses dépenses ainsi qu'avoir certaines statistiques sur son mois comme par exemple le montant total de dépense, revenus ou récurrence du mois. Elle dispose aussi d'une courbe montrant l'évolution de son solde sur les dernières transactions ainsi qu'un camenbert indiquant dans quelle catégorie nous avons effectué le plus de dépenses

Node js, Html, Css, mysql, Charts js

Mon serveur Apache

J'ai réalisé mon propre serveur web à l'aide d'un deuxième ordinateur installé chez moi. Ce PC est sous Ubuntu 20.04 avec docker. Il y a un premier conteneur proxy avec apache et un autre dans un réseau privé avec LAMP et node installé dessus. Pour rendre ce serveur web accessible par tout le monde sur internet j'ai dû faire la demande auprès de mon fournisseur internet d'une adresse ip fixe qui m'a ensuite permis de créer des redirections de ports depuis mon adresse ip vers mon serveur web.

Apache, Docker, Php, mysql, Node js, Npm

SkillTouch

Durant mon alternance au sein de CD2RI je travaille sur un prototype d'application de recherche d'emploi avec une utilisation type Tinder. Vous pouvez vous inscrire soit en tant qu'utilisateur soit en tant qu'employeur. Vous définissez ensuite votre recherche (stage, alternance, CDD, CDI) ainsi qu'une liste de savoir faire et savoir être vous correspondant ou correspondant à l'offre d'emploi que vous déposer en tant qu'employeur. Des profils défilleront ensuite basé sur vos critères de recherche et vous avez la possibilité de passer ou de liker le profil. Si un like a été générer des deux cêtés cela crée un match et l'utilisateur ainsi que l'employeur ont la possibilité de discuter pour faire connaissance et potentiellement définir un entretien.

GitHub lance son outil Copilot pour aider les développeurs

Github Copilot est une intelligence artificielle créée par GitHub et OpenAI dans le but d'aider les développeurs. Elle est capable de générer du code à partir d'un commentaire mais aussi de faire une prédiction de ce que vous voulez potentiellement écrire grace au modèle de Langages GPT3 développé par OpenAI.

Cette AI permet donc la prédiction dans une douzaine de Langages et s'installe sur certains IDE dont Visual Studio Code. GitHub Copilot est disponible par abonnement mensuel (9.99$) ou annuel (99$) mais dispose aussi d'une offre gratuite pour les étudiants Présenté le 29 juin 2021, GitHub Copilot est maintenant disponible pour les développeurs depuis le 20 juillet 2021. Afin d'apprendre à coder Github Copilot s'est entrainé sur plus de 1 milliard de lignes de code à travers tous les repositorys github disponibles.

Afin de valider la proposition de github il suffit de cliquer sur la touche TAB et de laisser github copilot faire son travail. Attention : Github Copilot est encore en phase de test et peut donc proposer des erreurs de syntaxe ou des erreurs de logique
